Erebot_Module_WatchList is a module for Erebot that can be used to handle a WATCH list, ie. track when some user (dis)connects.

The WATCH list is implemented using either the ISON command (see RFC 2812) or the WATCH (see http://docs.dal.net/docs/misc.html#4) extension. Exactly which mechanism is used depends on what the IRC server supports.
